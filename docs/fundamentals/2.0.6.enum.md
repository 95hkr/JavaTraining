### <a name="sec1"></a>列挙型とは
列挙型とは、プログラマが任意に作成できるある特定の値のみを持つ型のことくをいいます。  
列挙型を定義するには、enumキーワードを使用して型を宣言して任意の値を列挙します。

列挙型はクラス内でもクラス外でも宣言できますが、メソッドの中には宣言することはできません。

```sh
[修飾子] enum 列挙型名{値1,値2,値3,・・・n}
```
